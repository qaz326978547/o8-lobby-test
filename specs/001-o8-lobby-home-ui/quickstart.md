# 驗收快速入門：O8 手機版遊戲大廳首頁 UI Demo

**功能分支**：`001-o8-lobby-home-ui`
**日期**：2026-06-04

---

## 前置條件

- Node.js `^20.19.0` 或 `>=22.12.0`
- 已執行 `npm install`（包含 `sass-embedded` 與 `vue-i18n`）

---

## 啟動開發環境

```bash
npm run dev
```

開啟瀏覽器，前往 `http://localhost:5173`（或 Vite 顯示的實際 port）。

建議開啟 Chrome DevTools，切換至 **Device Toolbar**，選擇 iPhone 14 Pro（或設定自訂
寬度 390px）進行手機版驗收。

---

## 情境驗收清單

### US1 — 首頁預設狀態瀏覽

**設定**：DevTools 寬度設為 390px。

| 驗收項目 | 操作 | 預期結果 |
|----------|------|----------|
| 頁面完整載入 | 開啟首頁 | Header（Logo + 選單 + 語系）、Banner、導覽列、熱門遊戲、新品推薦、右下浮動按鈕均可見 |
| 版面接近參考圖 | 比對 `reference/o8-mobile-home-search-reference.png` 左側 | 比例、色彩、間距高度吻合 |
| 桌面置中驗收 | 將瀏覽器寬度拉至 1200px | `.container` 寬度 ≤ 1140px 並置中；AppHeader 與 HeroBanner 仍佔滿視口全寬 |
| 中間寬度無破版 | 將寬度設為 600px | 所有區塊不溢出、不重疊 |

---

### US2 — 搜尋面板開啟與關閉

| 驗收項目 | 操作 | 預期結果 |
|----------|------|----------|
| 面板開啟 | 點擊導覽列右側搜尋按鈕 | 半透明遮罩覆蓋頁面，搜尋面板出現於遮罩上方 |
| 搜尋框 placeholder | 開啟面板，zh-TW 狀態 | 顯示「遊戲、遊戲商、類型」 |
| Tags 顯示 | 開啟面板 | 顯示 ATG、AG Live、戰神賽特、電子（或 Slots） |
| 遮罩關閉 | 點擊遮罩區域 | 面板關閉，頁面恢復預設狀態 |
| 清除關閉 | 開啟面板後點擊清除文字 | 面板關閉，頁面恢復預設狀態 |
| 面板清晰可操作 | 開啟面板後嘗試點擊搜尋框 | 搜尋框可聚焦，不被遮罩遮蔽 |

### US3 — 語系切換

| 驗收項目 | 操作 | 預期結果 |
|----------|------|----------|
| 切換至 en-US | 點擊 Header 語系切換 icon（zh-TW → en-US） | 「發現」→ Discover、「熱門遊戲」→ Hot Games、「新品推薦」→ New Releases |
| 搜尋 placeholder 更新 | 切換語系後開啟搜尋面板 | placeholder 顯示 `Games, Providers, Categories` |
| 清除文字更新 | 切換語系後開啟搜尋面板 | 顯示 `Clear` |
| Slots 更新 | 切換語系後開啟搜尋面板 | 「電子」tag 顯示 `Slots` |
| 廠商名稱不翻譯 | 切換語系 | ATG、BT、DB、FC、US、AG Live 維持原文 |
| 遊戲名稱不翻譯 | 切換語系 | Gods-Of-Glory、Coins-And-Cannons 等維持原文 |
| 切換回 zh-TW | 再次點擊語系 icon | 所有文字恢復繁體中文 |

---

### US4 — 遊戲平台導覽列水平滑動

| 驗收項目 | 操作 | 預期結果 |
|----------|------|----------|
| 列表可滑動 | 390px 寬度下手指（或觸控板）向左滑動導覽列 | 列表水平滑動，顯示更多平台 |
| 整頁不橫向捲動 | 滑動導覽列時 | 頁面主體無水平捲軸或橫向位移 |
| 搜尋按鈕固定 | 滑動至任意位置 | 搜尋按鈕維持於右側可見且可點擊 |

---

### US5 — 遊戲卡片自動輪播

| 驗收項目 | 操作 | 預期結果 |
|----------|------|----------|
| 初始第 1 頁 | 開啟頁面 | 熱門遊戲與新品推薦各自顯示第 1 頁 4 張卡片（2×2），進度條指示器第 1 條為 active（金色）|
| 5 秒自動切換 | 靜候 5 秒 | 自動切換至第 2 頁，進度條第 2 條變為 active（fade 效果） |
| 循環回第 1 頁 | 再靜候 5 秒 | 回到第 1 頁，進度條第 1 條再次 active |
| 兩區塊獨立 | 觀察熱門遊戲與新品推薦 | 各自獨立計時，不同步切換（兩頁資料相同屬正常） |
| 卡片色碼驗收 | 檢視 GameCard-1 卡片 | footer 背景 #AE8640、紅色膠囊 #CE2727 |
| 綠色膠囊 | 檢視 Last-Man-Standing 卡片（值 70） | 膠囊顯示綠色 #35BD13 |
| 名稱截斷 | 確認 Coins-And-Cannons 卡片 | 名稱單行顯示，過長以省略號截斷，膠囊不被擠壓 |

---

### US6 — 回到頂部浮動按鈕

| 驗收項目 | 操作 | 預期結果 |
|----------|------|----------|
| 初始不顯示 | 開啟首頁（未捲動） | 右下方無回到頂部按鈕 |
| 捲動後顯示 | 向下捲動首頁內容 | 主內容容器右下方出現回到頂部按鈕 |
| 點擊回頂 | 點擊按鈕 | 頁面平滑捲動回頂部，按鈕隨即消失 |
| 按鈕不錯位 | 在 390px 與 768px 寬度下各自驗收 | 按鈕位置均在容器右下方，不超出容器外側 |


- `dist/` 目錄生成
- 開啟瀏覽器 console，確認無圖片載入失敗錯誤
